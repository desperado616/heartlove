# Быстрая инструкция по настройке

## Шаг 1: Установка зависимостей
```bash
pip install -r requirements.txt
```

## Шаг 2: Настройка конфигурации

Создайте файл `.env` в корне проекта со следующим содержимым:

```env
BOT_TOKEN=ваш_токен_от_BotFather
ALLOWED_CHAT_ID=ваш_chat_id
OPENROUTER_API_KEY=ваш_api_ключ_openrouter
```

### Как получить данные:

1. **BOT_TOKEN**: 
   - Откройте [@BotFather](https://t.me/BotFather)
   - Отправьте `/newbot` и следуйте инструкциям
   - Скопируйте токен

2. **ALLOWED_CHAT_ID**:
   - Откройте [@userinfobot](https://t.me/userinfobot)
   - Отправьте любое сообщение
   - Скопируйте ваш `Id`

3. **OPENROUTER_API_KEY**:
   - Зарегистрируйтесь на [OpenRouter.ai](https://openrouter.ai/)
   - Создайте API ключ в разделе "Keys"
   - Скопируйте ключ (начинается с `sk-or-v1-...`)

## Шаг 3: Запуск

```bash
python bot.py
```

Бот начнет работать и будет:
- Отвечать на сообщения пользователя
- Отправлять ежедневные напоминания в случайное время (10:00-20:00)
- Реагировать на триггерные слова в сообщениях
- Сохранять контекст диалога в SQLite

## Проверка работы

Отправьте боту команду `/start` или просто напишите любое сообщение. При первом сообщении бот отправит приветственное сообщение.
